# 介绍
xyz-components是基于vue2.0的组件库。

引用时可以

###直接引入js
```

<script src="path/to/xyz-components/dist/index.js"></script>
```
###通过npm安装

# 开发 
* `npm install`
* `npm run build`,打包压缩组件库到dist目录

# 新写组件
* 在`src/components`下新建组件
* 在`src/index.js`文件中，注册新组件
